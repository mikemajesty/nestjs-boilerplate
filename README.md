# NestJs boilerplate API

## Description

[Nest](https://docs.nestjs.com/) framework documentaion.

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ start:dev
```

```bash
# production
$ docker-compose up --build
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn  test:e2e

# test coverage
$ yarn  test:coverage
```
